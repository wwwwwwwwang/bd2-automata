## ADDED Requirements

### Requirement: 用户/角色/权限域提供统一 CRUD
系统 MUST 为用户、角色、权限资源提供统一接口能力：按 ID 查询单条、分页查询、新增、修改、删除。

#### Scenario: 用户资源 CRUD 可用
- **WHEN** 客户端访问用户资源接口
- **THEN** 可执行单条查询、分页查询、新增、修改、删除

#### Scenario: 角色资源 CRUD 可用
- **WHEN** 客户端访问角色资源接口
- **THEN** 可执行单条查询、分页查询、新增、修改、删除

#### Scenario: 权限资源 CRUD 可用
- **WHEN** 客户端访问权限资源接口
- **THEN** 可执行单条查询、分页查询、新增、修改、删除

### Requirement: 关联关系接口与删除策略一致
系统 MUST 对用户-角色、角色-权限等关联关系按关联表策略执行硬删除。

#### Scenario: 关联关系删除
- **WHEN** 删除某条用户-角色或角色-权限关联
- **THEN** 记录被物理删除
- **AND** 后续查询不再返回该关联
