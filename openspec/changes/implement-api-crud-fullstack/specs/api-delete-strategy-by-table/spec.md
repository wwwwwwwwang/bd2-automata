## ADDED Requirements

### Requirement: 删除语义按表区分
系统 MUST 按表类型执行删除策略：核心业务表软删除，关联/中间表硬删除。

#### Scenario: 核心业务表软删除
- **WHEN** 对核心业务表资源执行删除
- **THEN** 系统必须写入删除标记而非物理删除
- **AND** 默认查询不得返回已软删除记录

#### Scenario: 关联/中间表硬删除
- **WHEN** 对关联/中间表记录执行删除
- **THEN** 系统必须执行物理删除
- **AND** 删除后记录不可再被查询到

### Requirement: 默认查询行为一致性
系统 SHOULD 在未显式声明包含已删除数据时，统一过滤软删除数据。

#### Scenario: 普通查询默认过滤软删
- **WHEN** 客户端发起常规列表或详情查询
- **THEN** 返回结果中不包含软删除记录
